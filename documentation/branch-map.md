# Mapa de Ramas

Documento operativo para seguimiento de ramas activas y stale.

## Como completar este mapa
1. Obtener ramas locales y remotas.
2. Revisar ultimo commit por rama y evidencia de actividad.
3. Estimar funcion de la rama por nombre y cambios recientes.
4. Marcar estado: Activa, Stale o Incierta.

## Tabla de ramas

| Rama | Funcion estimada | Evidencia | Estado | Accion sugerida |
|---|---|---|---|---|
| main | Pendiente | Pendiente | Incierta | Revisar historial |

## Criterio para stale
Una rama puede marcarse como Stale si no presenta actividad reciente y no tiene integraciones activas conocidas.

## Regla de mantenimiento
Actualizar este archivo en cada ciclo de documentacion que incluya analisis de cambios o preparacion de release.
